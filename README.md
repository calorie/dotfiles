dotfiles
========

## Setup

```sh
git clone --single-branch git@github.com:calorie/dotfiles.git ~/dotfiles
cd ~/dotfiles
./script/setup
```

```
sudo ln -sfn /opt/homebrew/opt/openjdk@19/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-19.jdk
```

## Uninstall

```sh
cd ~/dotfiles
./script/uninstall
```
