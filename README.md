dotfiles
========
# Setup

```
$ git clone git://github.com/calorie/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ ./script/setup
```

# GoogleJapaneseInput

## キー設定

環境設定 > 一般 > キー設定の選択 編集 > 編集 > インポート

```
~/dotfiles/.config/GoogleJapaneseInput/keymap.txt
```

## 辞書登録

辞書ツール > 管理 > 選択した辞書にインポート

```
~/dotfiles/.config/GoogleJapaneseInput/dictionary.txt
```

# Uninstall

```
$ cd ~/dotfiles
$ ./script/uninstall
```
